# בלוג משחקי מחשב (Gaming Blog)

אתר בלוג למשחקי מחשב עם עמוד ראשי, עמודים דינמיים, פוסטים, פאנל ניהול ומערכת התחברות והרשמה. מחובר לשרת ומסד נתונים MySQL.

## מה כולל הפרויקט

- **עמוד ראשי** – דף בית עם פוסטים אחרונים וקישורים
- **עמודים** – עמודים סטטיים שניתן ליצור ולערוך (אודות, צור קשר וכו')
- **פוסטים** – פוסטים עם כותרת, תקציר, תוכן, תמונת כותרת ופרסום/טיוטה
- **פאנל ניהול** – דשבורד, ניהול עמודים ופוסטים (הוספה, עריכה, מחיקה)
- **התחברות והרשמה** – משתמשים יכולים להירשם ולהתחבר; גישה לפאנל למי שמחובר

## דרישות

- PHP 7.4+ (עם PDO ו־MySQL)
- MySQL 5.7+ או MariaDB
- שרת web (Apache/Nginx) או PHP built-in server

## התקנה וחיבור לשרת ומסד נתונים

### 1. העתקת הקבצים לשרת

העלה את תיקיית `gaming-blog` לשרת (או השאר מקומית לפיתוח).

### 2. יצירת מסד הנתונים

התחבר ל־MySQL והרץ את קובץ ה־schema:

```bash
mysql -u root -p < database/schema.sql
```

או ב־phpMyAdmin: ייבא את הקובץ `database/schema.sql`.

זה ייצור:

- מסד נתונים `gaming_blog`
- טבלאות: `users`, `pages`, `posts`, `categories`, `post_categories`
- משתמש מנהל ברירת מחדל: **admin** / **password** (חשוב להחליף סיסמה בפרודקשן!)

### 3. הגדרת חיבור למסד הנתונים

ערוך את `config.php` בהתאם לשרת ולמסד הנתונים:

```php
// מסד נתונים
define('DB_HOST', 'localhost');   // כתובת השרת
define('DB_NAME', 'gaming_blog'); // שם המסד
define('DB_USER', 'root');        // משתמש MySQL
define('DB_PASS', '');            // סיסמה

// כתובת האתר (לפי השרת שלך)
define('BASE_URL', 'http://localhost/gaming-blog');
```

בשרת אמיתי החלף את `BASE_URL` לכתובת האתר, למשל:

`https://yourdomain.com`

### 4. הרצת האתר

**מקומית עם PHP:**

```bash
cd gaming-blog/public
php -S localhost:8000
```

אז האתר יהיה בכתובת: `http://localhost:8000`

**ב־Apache/Nginx:** הגדר ש־Document Root מצביע על תיקיית `public` (או שכל האתר תחת תיקייה אחת והתאמת הנתיבים ב־config).

### 5. הרשאות (אם צריך)

ודא שלשרת יש הרשאה לכתוב בתיקייה אם תוסיף בהמשך העלאת קבצים (למשל `uploads/`).

```

├── config.php           # הגדרות + חיבור DB
├── database/
│   └── schema.sql      # יצירת טבלאות
├── includes/
│   ├── Auth.php        # התחברות/הרשמה
│   ├── Database.php    # חיבור PDO
│   ├── helpers.php     # פונקציות עזר
│   ├── Page.php        # מודל עמודים
│   └── Post.php        # מודל פוסטים
├── public/             # הממשק הציבורי
│   ├── index.php       # עמוד ראשי
│   ├── posts.php       # ארכיון פוסטים
│   ├── post.php        # פוסט בודד
│   ├── page.php        # עמוד בודד
│   ├── login.php       # התחברות
│   ├── register.php    # הרשמה
│   ├── logout.php      # התנתקות
│   └── layouts/
├── admin/              # פאנל ניהול
│   ├── index.php       # דשבורד
│   ├── pages/          # ניהול עמודים
│   └── posts/          # ניהול פוסטים
└── assets/
    ├── css/
    └── js/
```

## שימוש

1. **צפייה באתר:** כניסה לכתובת האתר (עמוד ראשי, פוסטים, עמודים).
2. **הרשמה:** קישור "הרשמה" → מילוי פרטים → לאחר מכן "התחברות".
3. **התחברות:** משתמש **admin** וסיסמה **password** (ברירת מחדל) או משתמש שנרשם.
4. **פאנל ניהול:** אחרי התחברות – "פאנל ניהול" או כניסה ל־`/admin/` – משם ניהול עמודים ופוסטים.

## אבטחה (פרודקשן)

- החלף את סיסמת המשתמש **admin** מיד אחרי ההתקנה.
- ב־`config.php` כבה הצגת שגיאות: `display_errors = 0`, `error_reporting` בהתאם.
- השתמש ב־HTTPS.
- ודא שהתיקייה `database/` לא נגישה מהדפדפן (מחוץ ל־public).
